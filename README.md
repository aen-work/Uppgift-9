HTML and CSS school assignment 9 in the course "HTML, CSS och tillgänglighet".
We practised how to recreate a login page from a design sketch into HTML and CSS code from scratch.
Using CSS flexbox and responsive technologies in VS code editor.

Course Purpose:
The purpose is to provide knowledge about HTML and CSS, how these are used to present and
declare the content on web pages, and how web pages are adapted to different conditions
through responsive design and accessibility technology.

Course Objectives:
The objective is to code in HTML and CSS and convert a design sketch to HTML and CSS. The
student should have a good understanding of how HTML and accessibility are related, how media
queries are used, application of mobile first as a method and a basic understanding of
how HTML and SEO are related. The student should be able to create layouts and have knowledge of layout models, such as float, flexbox and grid, their purpose and interaction.
The objective is to understand the further development of the web platform, create and analyze websites
from an accessibility perspective, know guidelines and standards and why they are important from a
business and ethical perspective.
